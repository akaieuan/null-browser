# Screenshots

Drop PNG screenshots in this directory and reference them from the top-level
`README.md`. The top-level README expects:

- `overview.png` — full window with a page loaded and the Profile panel
  open (appearance swatches, start page, search engine, AI providers).

Suggested capture style:
- 1440×900 logical (native Retina → 2880×1800 physical). Use macOS `⌘⇧4`
  then `Space` then click the window for a clean window-only PNG with the
  macOS shadow.
- Dark palette (Neutral or Slate) for consistency with the PHILOSOPHY doc.
- No identifiable personal content in the address bar or history.

Add more as the project grows (Network Inspector view, History panel,
Settings, bookmarks bar with real bookmarks, etc.) and wire them in from
the appropriate README section.
